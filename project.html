校園數位分身 VR (IEEE ECICE)

開發虛擬實境校園導覽系統，獲得 IEEE ECICE 最佳會議論文獎。

  • Immersive Campus Exploration: VR-Driven Navigation Solutions(歡迎搜尋)
  • 精確建模中原大學校園環境，包括建築物、景觀和內部設施
  • 實現沉浸式虛擬導覽體驗，允許用戶自由探索校園
  • 整合互動式信息展示功能，提供校園歷史、設施介紹等
  • 開發多平台兼容性，支持VR設備和普通電腦瀏覽

Sidelobe Reduction (IEEE ICASI)

以新架構天線設計，在不調整訊號處理的情況下，優化Farfield，降低旁瓣。

  • Sidelobe Level Reduction in Uniform Planar Phased Arrays Through Feed Position Arrangement(歡迎搜尋)
  • 使用CST Studio Suite進行天線建模和電磁場仿真
  • 設計並優化4x4 MIMO天線陣列
  • 通過結構調整實現旁瓣抑制，提高主瓣增益
  • 分析並比較優化前後的輻射方向圖和S參數

MR 機械手臂開發 (MetaACES)

利用 HoloLens 2 開發機械手臂操作教學系統，實現虛擬與現實的無縫結合。

  • 應用 HoloLens 2 的混合實境技術，創建虛擬機械手臂模型
  • 開發直覺式的機械手臂操作界面，支持手勢和語音控制
  • 實現實時互動和操作反饋，提供即時的視覺和聽覺指導
  • 設計循序漸進的學習模塊，從基礎操作到複雜任務

8x8 MIMO 手機天線結合機器學習

利用機器學習技術優化8x8 MIMO手機天線設計，自動尋找最佳天線間距。

  • 開發機器學習模型，預測不同天線間距的性能表現
  • 大幅優化S參數和farfield性能，提高天線效率
  • 成功將天線的包絡相關係數（ECC）降低50%
  • 建立自動化數據收集和處理流程，生成全面的訓練數據集

Delta台達實習-深度影像辨識ROI

開發深度學習模型，用於檢測電路板等零組件的瑕疵。

  • 設計和實現基於深度學習的影像辨識系統
  • 開發自動化ROI（感興趣區域）檢測算法
  • 優化模型以提高瑕疵檢測的準確率和效率
  • 實現與生產線系統的整合，實現實時檢測
  • 建立自動化數據收集和處理流程，生成全面的訓練數據集

Toyota 人因工程影像辨識

為Toyota開發基於影像辨識的人因工程分析系統，優化工作環境和流程。

  • 設計影像捕捉和分析系統，評估工作姿勢和動作
  • 開發機器學習模型，自動識別潛在的人體工學風險
  • 創建可視化工具，直觀展示分析結果和改進建議
  • 與Toyota工程團隊合作，將系統整合到生產環境中

股價預測軟體開發

開發了一個基於 Facebook Prophet 的股價預測軟體，利用時間序列分析來預測未來股價走勢。

  • 採用 Facebook Prophet 库進行時間序列預測，適應股價的季節性和趨勢變化
  • 使用 Python 和 Pandas 處理和分析大量歷史股價數據
  • 整合額外的外部因素（如市場指數、經濟指標）以提高預測準確性
  • 開發交互式可視化界面，使用 Plotly 或 Matplotlib 展示預測結果和置信區間
  • 實現自動化數據獲取和模型更新流程，確保預測的時效性
  • 進行交叉驗證和回測，評估模型在不同時間段的表現
